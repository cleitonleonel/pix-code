Pra pagar ou receber: manda um Pix-code!

Crie um (ou vários) links Pix-code e compartilhe com seus amigos e clientes. Eles poderão ver os detalhes da fatura, um QR Code para pagamento além do código "Pix Copia e Cola".
Atenção: a Chave Pix deve ser cadastrada diretamente no aplicativo do seu banco.
Atenção

Alguns bancos recebedores não exibem nos comprovantes o Identificador (cuja acessibilidade o BACEN só exige nas consultas feitas à API Pix do banco) nem a mensagem digitada pelo pagador (informação que é obrigatória mas os bancos ainda estão se adequando a essa normatização).

Atenção

A consulta do proprietário da Chave Pix é feita pelo banco do pagador junto ao BACEN mas o fornecimento desse dado é útil para a exibição na página de fatura.

Como funciona?

As informações fornecidas serão codificadas e um link será gerado. Quando o link for acessado, uma página de cobrança será carregada em tempo real (não armazenamos as informações).

Funcionalidades

  • Geração de QR Code
    Conforme documentação Pix
  • Página de pagamento
    com QR Code e "Pix Copia e Cola"
  • Consumo via API
    curl -sX POST https://pix-code.herokuapp.com
  • Faturas com sua marca
    Entre em contato
